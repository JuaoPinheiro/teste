# Teste 


Este projeto foi desenvolvido com a intenção de demonstrar um exemplo de um consultor de CPF em formato de teste. Basicamente, ele obtém o número do CPF de um banco de dados fictício e simula a quantidade de dívidas que o cliente possui, considerando possíveis descontos. Além disso, o projeto oferece a opção de entrar em contato com uma empresa que presta assistência nessa área.
### Link do projeto: <a href='https://teste-theta-one.vercel.app'> Aqui!</a>

# 👨‍💻 Instalação

```bash
# Clone this project
$ git clone https://github.com/JuaoPinheiro/teste
# Access
$ cd teste
# Install dependencies
$ yarn
# Run the project
$ yarn dev
# The server will initialize in the <http://localhost:3000>
```
